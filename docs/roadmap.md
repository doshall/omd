# 路线图

本文档记录 omd 的未来规划。当前 v0.1.0 已具备完整的 Markdown 编辑和预览能力，以下为可选的增强方向。

## 当前状态（v0.1.0）

### 已完成

- [x] 桌面版原生 GUI 编辑器
- [x] Web 版浏览器编辑器
- [x] 实时 Markdown 预览
- [x] 文件新建/打开/保存（桌面）/ 导入下载（Web）
- [x] 格式化工具栏
- [x] 深色/浅色主题
- [x] 图片支持（URL、本地、Base64、粘贴、拖拽）
- [x] Mermaid 图表（Web 版）
- [x] 任务列表、表格、删除线
- [x] 移动端响应式布局（Web 版）
- [x] 自动保存（Web 版 localStorage）
- [x] 应用状态持久化（桌面版）
- [x] 完整项目文档

## 近期计划

优先级较高、投入产出比较好的功能：

| 功能 | 版本 | 优先级 | 说明 |
|------|------|--------|------|
| 查找/替换 | 双版本 | 高 | `Ctrl+F` / `Ctrl+H` |
| 桌面版 Mermaid | 桌面 | 高 | 集成 mermaid 渲染 |
| 导出 HTML | 双版本 | 中 | 将 Markdown 导出为独立 HTML |
| GitHub Pages CI | Web | 中 | 自动部署流水线 |
| PWA 支持 | Web | 中 | 可安装到手机主屏幕 |
| 代码块语法高亮 | 双版本 | 中 | 预览区着色 |

## 中期计划

提升编辑体验的功能：

| 功能 | 版本 | 说明 |
|------|------|------|
| 多标签页 | 双版本 | 同时编辑多个文档 |
| 最近文件列表 | 桌面 | 快速打开历史文件 |
| 自动保存到磁盘 | 桌面 | 定时保存，防丢失 |
| 编辑区语法高亮 | 双版本 | 编辑时 Markdown 着色 |
| 同步滚动 | 双版本 | 编辑区与预览区联动 |
| 导出 PDF | 双版本 | 一键导出 PDF |
| LaTeX 数学公式 | 双版本 | KaTeX / MathJax 集成 |
| 目录自动生成 | 双版本 | 基于标题生成 TOC |
| 桌面版粘贴图片 | 桌面 | 剪贴板图片插入 |
| IndexedDB 存储 | Web | 突破 localStorage 限制 |

## 长期计划

更复杂或需要架构调整的功能：

| 功能 | 说明 |
|------|------|
| 插件系统 | 允许第三方扩展 |
| Git 集成 | 版本控制、diff 视图 |
| 云同步 | iCloud / WebDAV / 自建同步 |
| 多人协作 | 实时协同编辑 |
| Vim/Emacs 键位 | 编辑器模式切换 |
| 自定义主题 | 用户自定义 CSS / 配色 |
| 国际化 | 多语言界面 |
| 命令行接口 | `omd file.md` 直接打开 |
| 拼写检查 | 多语言拼写 |

## 不考虑的功能

以下功能不在规划范围内，以保持 omd 的轻量定位：

- 所见即所得（WYSIWYG）编辑模式
- 内置文件管理器 / 文件树
- 博客发布 / CMS 功能
- 数据库集成
- 重量级插件市场

## 如何影响路线图

1. 在 [GitHub Issues](https://github.com/doshall/omd/issues) 提交功能请求
2. 为现有 Issue 点赞（👍）表达需求优先级
3. 直接提交 PR 实现功能

参见 [贡献指南](../CONTRIBUTING.md)。

## 版本规划

| 版本 | 主题 | 预计内容 |
|------|------|----------|
| v0.2.0 | 编辑增强 | 查找替换、语法高亮、导出 HTML |
| v0.3.0 | 平台完善 | PWA、桌面 Mermaid、GitHub Actions CI |
| v0.4.0 | 体验提升 | 多标签页、PDF 导出、LaTeX |
| v1.0.0 | 稳定版 | API 稳定、完整文档、多平台发布包 |

> 版本号为规划参考，实际发布时间取决于贡献进度。
