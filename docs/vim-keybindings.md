# Vim 键位参考

omd 在 **设置 → 键位模式 → Vim** 下提供简化版 Vim 键位。桌面版与 Web/Android 版行为基本一致。

## 设置项

| 设置 | 默认 | 说明 |
|------|------|------|
| `vim_show_block_highlight` | 开启 | Visual Block 选区列块高亮 |
| `vim_use_system_clipboard` | 开启 | `"` / 小写命名寄存器与 `"+` / `"*` 同步系统剪贴板 |

## 模式

| 模式 | 进入 | 退出 |
|------|------|------|
| NORMAL | 默认 / `Esc` | `i`/`a`/`o` 等进入插入 |
| INSERT | `i` `a` `o` `O` `A` `I` | `Esc` |
| VISUAL | `v` + 移动 | `Esc` / `y` / `d` |
| VISUAL BLOCK | `Ctrl+V` + `hjkl` | `Esc` / `y` / `d` |
| COMMAND | `:`（桌面 `Shift+;`） | `Esc` / `Enter` |

状态栏显示当前 Vim 子模式、数字前缀与宏录制 `REC @a`。

## 寄存器

| 寄存器 | 说明 |
|--------|------|
| `"` | 无名寄存器（默认 yank/paste） |
| `"a`–`"z` | 命名寄存器 |
| `"0` | 最近一次 yank 的全文 |
| `"+` / `"*` | 系统剪贴板（需开启 `vim_use_system_clipboard`） |
| `"_` | 黑洞寄存器（丢弃） |

用法：`"ayy` 将当前行存入 `a`；`"ap` 粘贴；`:reg` 列出内容。

## Ex 命令（命令行模式）

| 命令 | 作用 |
|------|------|
| `:w` | 保存 |
| `:q` | 提示退出方式 |
| `:wq` / `:x` | 保存 |
| `:42` | 跳转到第 42 行 |
| `:5d` | 删除第 5 行 |
| `:1,5d` | 删除第 1–5 行 |
| `:g/TODO/d` | 删除包含 `TODO` 的行 |
| `:g/foo/s/foo/bar/g` | 在匹配行上将 `foo` 替换为 `bar` |
| `:g/^#/norm I- ` | 在匹配行首插入 `- `（norm 支持 `dd` `>>` `<<` `x` `dw` `cw` `ciw` `cc` `J` `o` `O` `rX`、数字前缀与 `I`/`A` 文本） |
| `:v/foo/d` | 删除不包含 `foo` 的行（同 `:g!/foo/d`） |
| `:%s/old/new/g` | 全文替换 |
| `:s/old/new` | 当前行替换一次 |
| `:set number` / `:set nonumber` | 开关行号 |
| `:reg` | 查看寄存器 |
| `:help` | 简要命令列表 |

## 常用 Normal 模式

- **移动**：`hjkl` `w` `b` `e` `0` `$` `gg` `G` `f`/`F`/`t`/`T` `;` `,`
- **删除/修改**：`x` `dd` `dw` `cw` `cc` `D`（可加数字前缀，如 `3dd`）
- **复制/粘贴**：`yy` `yw` `p` `P`
- **缩进**：`>>` `<<`
- **重复**：`.` 重复上一次修改；`@@` 重复上一次宏
- **宏**：`qa` 开始录制到寄存器 `a`，`q` 停止，`@a` 回放

## Visual Block

1. `Ctrl+V` 进入块选择
2. `hjkl` 扩展矩形选区（列块会高亮显示）
3. `y` 复制列块到寄存器，`d`/`x` 删除列块，`c` 删除列块并进入块插入，`~` 切换列块大小写
4. `>` / `<` 对列块所在行缩进 / 反缩进
5. `I` / `A` 在列块前/后进入块插入模式（输入同步到所有行）

## Emacs 模式（扩展）

| 快捷键 | 作用 |
|--------|------|
| `Ctrl+Space` | 设置标记（mark） |
| `Ctrl+G` | 取消前缀 / 清除标记 |
| `Ctrl+O` | 在下方打开新行 |
| `Ctrl+J` | 与下一行合并 |
| `Ctrl+T` | 交换光标前两个字符 |
| `Alt+W` | 复制区域到剪切环（不删除） |
| `Ctrl+S` | 增量搜索（`Enter` 确认，`Ctrl+G`/`Esc` 取消，`Ctrl+S` 下一个；匹配文本高亮） |
| `Ctrl+R` | 反向增量搜索（匹配文本高亮） |
| `Alt+<` / `Alt+,` | 跳到缓冲区开头 |
| `Alt+>` / `Alt+.` | 跳到缓冲区末尾 |
