# 三端功能对比

本文档对比 omd **桌面版**、**Web 版**与 **Android 版**（当前 **v0.9.x**），便于选择合适版本。

## 总览

| 维度 | 桌面版 | Web 版 | Android 版 |
|------|--------|--------|------------|
| 安装 | 二进制 / 自行编译 | 零安装（浏览器） | 安装 APK |
| 平台 | Linux / macOS / Windows | 现代浏览器 | Android 8.0+（API 26+） |
| 技术栈 | egui + Rust | Leptos + WASM | WebView + 同 Web WASM |
| 离线 | ✅ 完全离线 | ✅ PWA / 自托管可离线 | ✅ APK 内打包资源 |
| 启动 | &lt;1 秒 | 1–3 秒（WASM 加载） | 约 1–2 秒 |
| 手机 | ❌ | ✅ 响应式 | ✅ 原生 APK |

## 使用场景

| 场景 | 推荐 |
|------|------|
| 日常本地写作、可靠保存 | **桌面版** |
| 浏览器即用、团队内网部署 | **Web 版** |
