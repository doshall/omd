# 版本功能对比

本文档详细对比 omd 桌面版与 Web 版的功能差异，帮助选择合适版本。

## 总览

| 维度 | 桌面版 | Web 版 |
|------|--------|--------|
| 安装 | 需编译或下载二进制 | 零安装，浏览器打开 |
| 平台 | Linux / macOS / Windows | 任意现代浏览器 |
| 手机 | ❌ | ✅ 响应式布局 |
| 离线 | ✅ 完全离线 | 部署后可离线（Mermaid 需 CDN 或本地化） |
| 性能 | 原生，流畅 | WASM，首次加载 1–3 秒 |

## 技术对比

| 维度 | 桌面版 | Web 版 |
|------|--------|--------|
| 语言 | Rust | Rust → WASM |
| GUI | egui (即时模式) | Leptos (声明式) |
| 渲染 | egui 原生控件 | HTML + CSS |
| Markdown 解析 | pulldown-cmark Event 流 | pulldown-cmark HTML |
| 二进制大小 | ~15–25 MB | WASM ~1–2 MB + JS |
| 启动时间 | <1 秒 | 1–3 秒（含 WASM 加载） |
| 内存占用 | ~50–100 MB | ~30–80 MB |
| 构建工具 | cargo | cargo + trunk |

## 使用场景推荐

### 选择桌面版

- 每天写作，管理本地 `.md` 文件
- 需要可靠的文件保存（`Ctrl+S`）
- 引用本地图片目录中的图片
- 完全离线环境
- Linux/macOS/Windows 桌面

### 选择 Web 版

- 手机或平板浏览器
- Cursor 云端 / 无 GUI 的远程环境
- 快速预览 Markdown，无需安装
- 需要 Mermaid 图表
- 需要粘贴截图
- 部署为团队内部工具

### 两者结合

- 桌面版日常写作 → 保存 `.md` 到磁盘
- Web 版随时预览/分享 → 打开同一 `.md` 文件
